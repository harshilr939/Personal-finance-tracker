# Personal Finance Tracker

A Python-based Personal Finance Tracker that helps users record, manage, and visualize their financial transactions.  
This project allows users to add income and expenses, view transaction history, calculate balances, and display expense charts.

## Features

- Add income and expense transactions
- View complete transaction history
- Automatically calculate account balance
- Visualize spending using charts
- Simple command-line interface
- Organized modular Python code

## Technologies Used

- Python
- Matplotlib (for charts)
- CSV / File Handling
- Object-Oriented Programming
